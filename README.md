# Identity & Access Management Portfolio

This repository demonstrates hands-on experience and design-led understanding of Identity and Access Management (IAM) using Microsoft Entra ID, Microsoft 365, and Azure.

The focus is on real-world access control, authentication, security posture, and operational troubleshooting rather than infrastructure-heavy labs.

---

## Core Skills Demonstrated
- Identity lifecycle management (Joiner / Mover / Leaver)
- Authentication & MFA enforcement
- Conditional Access & Security Defaults
- Privileged access & break-glass design
- Azure RBAC & identity-based access
- Microsoft 365 access troubleshooting
- Hybrid identity concepts
- Identity incident response
